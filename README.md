# Monocle Design System Study
>Web Development Design System Study

## About
This is a Case Study of the Design System from the Monocle news site. The project includes a preliminary breakdown and analysis, a component library, and a recreation of the homepage. This analysis was created as a final project for the Web Development course at MCAD.

-   [Preliminary Design Study Document](https://github.com/kpitman-mcad/monocle-casestudy/blob/main/assets/Pitman_Kevin_WebSystemStudy_Final.pdf)
-   [Component Library (in progress)](https://kpitman-mcad.github.io/monocle-casestudy/components/)
-   [Homepage Recreation (in progress)](https://kpitman-mcad.github.io/monocle-casestudy/)
